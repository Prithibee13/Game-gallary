Welcome to Rock-Paper-Scissors Game


How's it Work

There are three element into this game. 1) Rock, 2) Paper, 3) Scissors.

The flow-chart in the picture can describe overcomes of the game. An element can win only over it's anti-clockwise adjacent/nagibour element. Othwerwise it will be lose. If computer and player choose same element, the round will draw.

For win, winner get 3 points, losser gets 0 points, for draw round each player gets 1 points.
